# 游戏规则
- 顶部为生命条，会缓慢减少，需要持续吃小球续命。
- 吃到💊可以快速增加生命条，没事可以多吃点。
- 吃到💣会直接爆炸，游戏结束。
- 吃到小黄球会被打回原形，不要涉黄！
